# house-price-prediction
# House Price Prediction using Linear Regression

## Overview
This project predicts house prices based on:
- Square Feet
- Number of Bedrooms
- Number of Bathrooms

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Dataset

| Square Feet | Bedrooms | Bathrooms | Price |
|------------|-----------|------------|--------|
| 1500 | 3 | 2 | 250000 |
| 2000 | 4 | 3 | 350000 |
| 1200 | 2 | 1 | 180000 |
| 1800 | 3 | 2 | 300000 |

## How to Run

```bash
pip install pandas scikit-learn matplotlib
python house_price_prediction.py
```

## Sample Output

```text
Predicted House Price: 400000
R2 Score: 0.95
Mean Squared Error: 12000000
```
